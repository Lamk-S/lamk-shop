Un sistema integral de punto de venta (POS) y control de inventarios diseñado para la gestión ágil de tiendas de ropa, calzado y accesorios deportivos. Optimizado para un flujo de caja rápido y operaciones de alto volumen mediante soporte avanzado de escáneres de códigos de barras.

## ✨ Características Principales

* **Ventas Ágiles (POS):** Interfaz fluida para cajeros, compatible con escaneo de qr o código de barras. Soporta pagos mixtos (Efectivo, Tarjeta, Yape, Plin) y creación de clientes en tiempo real.
* **Catálogo Multidimensional:** Gestión estructurada de productos con soporte para variantes (SKUs). Organización por Líneas (Categorías), Fabricantes (Marcas) y Calibres (Tallas exclusivas para calzado, ropa o accesorios).
* **Control de Stock Reactivo:** Alertas automáticas de nivel de inventario y prevención de sobreventa en caja.
* **Auditoría y Trazabilidad:** Módulo de seguridad que registra cada movimiento (creación, edición, eliminación) con estampa de tiempo, usuario responsable e IP, garantizando transparencia.
* **UX/UI Optimizada:** Interfaz "Soft Retail" construida con Bootstrap 5, enfocada en la reducción de clics mediante búsquedas asíncronas y debouncing (AJAX).

## 🚀 Tecnologías Utilizadas

* **Backend:** Laravel (PHP 8.x)
* **Frontend:** Blade, Bootstrap 5, CSS3 Custom
* **Base de Datos:** MySQL / MariaDB
* **Interactividad:** JavaScript, jQuery, SweetAlert2, Bootstrap Select

## 📋 Requisitos Previos

Asegúrate de tener instalados los siguientes componentes en tu entorno local o servidor:

* PHP >= 8.1
* Composer
* MySQL >= 5.7 o MariaDB >= 10.3
* Node.js y NPM (Opcional, para compilación de assets)
